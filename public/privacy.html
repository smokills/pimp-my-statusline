Privacy

Last updated: June 7, 2026

The short version: no cookies, no accounts, no personal data stored on any server. Everything you build stays in your browser.

Your work stays on your device

The statusline configuration you build is saved in your browser's localStorage so it survives a refresh. It is never sent anywhere. Clearing your browser data for this site removes it.

Exported scripts

Scripts are generated entirely in your browser and run on your machine. They make no network calls and contain no telemetry: they read the session JSON Claude Code pipes in and print your statusline, nothing else.

Anonymous usage statistics

This site uses GoatCounter, a privacy-friendly analytics service, to count visits. It records the page URL, the referring site, browser type, screen size and the country derived from your IP address. It sets no cookies, builds no profile, and does not store your IP address. See the GoatCounter privacy policy.

The legal basis for this processing is legitimate interest (art. 6(1)(f) GDPR): knowing how many people visit and where they come from, in aggregate.

Hosting

The site is served by GitHub Pages. Like any web host, GitHub may log requests (including IP addresses) for security and operational purposes. See the GitHub Privacy Statement.

Contact

Pimp My Statusline is a free, open-source project. For any question about this policy, open an issue on GitHub.
